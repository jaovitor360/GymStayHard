# 🏋️ GYM Stay Hard

## 📖 Sobre o Projeto

O **GYM Stay Hard** é um projeto acadêmico desenvolvido como trabalho final da disciplina de Desenvolvimento Web. O objetivo é apresentar um site voltado para o universo da musculação, oferecendo uma interface simples, responsiva e organizada para usuários que desejam conhecer a academia, visualizar treinos e acompanhar informações relacionadas à sua evolução física.

O projeto foi desenvolvido utilizando apenas **HTML5** e **CSS3**, aplicando conceitos de estrutura semântica, responsividade, Flexbox e Grid Layout.

---

## 🎯 Objetivo

Desenvolver um front-end funcional para um sistema web de academia, colocando em prática os conhecimentos adquiridos durante a disciplina.

---

## 🚀 Funcionalidades

* Página inicial com apresentação da academia.
* Menu de navegação entre as páginas.
* Área de login.
* Área de cadastro de usuários.
* Página de treinos.
* Página de evolução física.
* Página de peso corporal.
* Página de calendário semanal de treinos.
* Seção de contato no rodapé.
* Layout responsivo para computadores, tablets e celulares.

## 📸 Páginas do Sistema

### 🏠 Home

Apresenta a academia, seus principais recursos e informações sobre os serviços oferecidos.

### 🔐 Login

Permite que o usuário acesse sua conta.

### 📝 Cadastro

Formulário para criação de uma nova conta utilizando validações do HTML5.

### 💪 Treinos

Exibe uma ficha de treino organizada por grupos musculares.

### 📈 Evolução

Mostra um exemplo de acompanhamento da evolução física do usuário.

### ⚖️ Peso Corporal

Apresenta informações como peso atual, altura, IMC e histórico de peso.

### 📅 Calendário

Exibe um planejamento semanal de treinos.
